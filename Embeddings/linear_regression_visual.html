📈 Linear Regression Complete Guide

What is Linear Regression?

Linear regression finds the best-fit straight line through data points to predict outcomes.

Real-world examples:
  • Predict house prices based on size
  • Estimate salary based on years of experience
  • Forecast sales based on advertising spend

📐 The Linear Equation

y = mx + b
Where:
y = predicted value (dependent variable)
x = input value (independent variable)
m = slope (how steep the line is)
b = y-intercept (where line crosses y-axis)
Alternative notation: ŷ = β₀ + β₁x
ŷ (y-hat) = predicted value
β₀ (beta-0) = intercept
β₁ (beta-1) = slope coefficient

🧮 Key Formulas

1. Slope (m)

m = Σ[(xᵢ - x̄)(yᵢ - ȳ)] / Σ[(xᵢ - x̄)²]
Or using covariance: m = Cov(x,y) / Var(x)

2. Intercept (b)

b = ȳ - m·x̄
Where x̄ and ȳ are the means of x and y

3. Mean Squared Error (MSE)

MSE = (1/n) Σ(yᵢ - ŷᵢ)²
Measures average squared difference between actual and predicted values

4. R² Score (Coefficient of Determination)

R² = 1 - (SS_res / SS_tot)
R² = 1 → Perfect fit
R² = 0 → No better than mean
Range: 0 to 1 (higher is better)

📊 Residuals Visualization

Residuals = Actual value - Predicted value
Red lines show the error for each point. We minimize the sum of squared residuals!

🔍 Step-by-Step Process

Step 1: Collect Data

Gather (x, y) pairs of observations

Step 2: Calculate Means

Find x̄ (mean of x) and ȳ (mean of y)

Step 3: Calculate Slope

Use the slope formula with covariance

Step 4: Calculate Intercept

Use b = ȳ - m·x̄

Step 5: Make Predictions

Use ŷ = mx + b for new values

Step 6: Evaluate

Calculate MSE and R² to assess fit
